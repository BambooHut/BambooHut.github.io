Toepassingslaag

Apps

Het woord App is een afkorting van "applicatie", letterlijk: toepassing. Het is een programma die dan gedownload op een apparaat, zoals een computer of telefoon. Dit kunnen spelletjes zijn, maar kunnen ook bijvoorbeeld tekstverwerkers zijn, zoals Microsoft word. Deze programma's worden uiteindelijk door jouw computer weergegeven. Zie fysieke laag.

API's

API staat voor Application programming interface. Dit zorgt ervoor dat twee programma's (of apps) onderling kunnen communiceren. Zo kan jouw agenda app precies weten wanneer het de volgende dag is door een connectie met je klok.

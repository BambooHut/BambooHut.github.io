De logische laag is..... Een operating system (OS) bepaalt hoe de software van een apparaat de hardware van dat apparaat gebruikt. Het kan bijvoorbeeld andere programma's (software ) doawnloaden en in het interne geheugen stoppen, het besluit welke programma's door de CPU worden behandeld en kan de randapperatuur besturen. De hele reden dat je computer kan multitasken is omdat het OS razendsnel tussen programma's kan wisselen die dan worden behandeld door de CPU.

Drielagen model

Een apparaat wordt in 3 delen ingedeelt: de fysieke laag (de hardware), de logische laag (software die harware aanstuurt) en de toepassingslaag (overige software, zoals apps).

Interfase

Interfaces connecteren de 3 lagen. Er zijn 3 soorten:

Hardware interface

Bij een hardware interface worden de fysieke componenten van een computer geconnecteerd. Ook worden de logische laag en de fysieke laag zo verbonden.

Software interfase

Software interfases verbinden onderling de software. Ze worden ook wel Application Programming Interfases (API) genoemd. Een API wordt ook gebruikt om verbinding te maken met het internet.

User interfaces

De user interfaces zorgen ervoor dat de gebruiker met het apparaat kan communiceren. Dit gaat meestal via een Graphic User Interface (GUI). Denk hierbij bijvoorbeeld aan knoppen, tekst, of de touchfunctie van een touch screen.
